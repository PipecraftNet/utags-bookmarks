# UTags 书签管理器

![UTags 徽章](https://img.shields.io/badge/status-活跃中-brightgreen)
![版本](https://img.shields.io/badge/版本-1.0.0-blue)

## ✨ 功能特性

- 多维度标签管理书签
- 智能搜索与复合过滤
- 可视化数据统计
- 多主题支持（亮色/暗色）
- 本地持久化存储

## 🚀 快速入门

```bash
# 克隆仓库
git clone https://github.com/utags/utags.git

# 安装依赖
cd utags && npm install

# 启动开发服务器
npm run dev
```

## 📦 生产构建

```bash
npm run build
```

## 🛠️ 开发指南

### 代码规范

- 使用 Prettier 自动格式化代码
- 提交前运行 `npm run lint` 检查代码规范

### 测试流程

```bash
# 单元测试
npm run test

# 交互式测试界面
npm run test:ui
```
